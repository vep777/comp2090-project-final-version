# COMP2090SEF Final Project Package

This repository contains the final version of the COMP2090SEF project.

## Structure
- `Task1_ParcelLockerSystem/` - OOP-based application
- `Task2_Heap_RadixSort/` - self-study data structure and algorithm
- `reports/` - final Word and PDF reports

## Task 1
Campus Parcel Locker Management System

Main features:
- Add customer
- Register parcel and assign locker
- Pick up parcel
- Search parcel
- Show locker status
- Save data / Load data

## Task 2
Self-study topics:
- Min Heap
- Radix Sort
