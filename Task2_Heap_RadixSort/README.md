# Task 2 - Heap and Radix Sort

## Overview
This folder contains the self-study part of the project. The chosen new data structure is Min Heap, and the chosen new algorithm is Radix Sort.

## Files
- `heap.py` - Min Heap implementation
- `radix_sort.py` - LSD Radix Sort implementation for non-negative integers
- `demo.py` - demonstration program

## Final improvements
- Added clearer heap demonstration
- Added `build_heap()` demonstration
- Added radix sort step-by-step output

## User Guide
1. Open a terminal in this folder.
2. Run the demo with:

```bash
python demo.py
```

3. The program will show:
   - heap insertion and extraction
   - heap construction by `build_heap()`
   - radix sort intermediate steps and final result
